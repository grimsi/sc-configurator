Cache Shadows:

Toggles shadow caching. Can result in higher performance.

Silhoutte:

This feature adds more detail to some objects at the cost of a few FPS.

Multi-GPU:

Toggles Multi-GPU support.

Navigation-System Multithreading:

Toggles own thread for navigation.

Particle Multithreading:

Toggles own thread for particle calculations.

Sub-Object Mesh Multithreading:

Toggles own thread for sub-object mesh calculations.

Threaded Rendering while Loading:

Toggles threaded rendering while the game is loading.
Number of Physics-Threads: Sets the number of internal physics-threads. Higher number can result in more performance but will be less stable.

Job System:

Toggles wether to start new jobs in own threads (disabled) or worker threads (enabled).
Index of Main-Thread Core: Sets the core on which the main thread runs.
Index of Physics-Thread Core: Sets the core on which the physics thread runs.
Index of I/O-Thread Core: Sets the core on which the i/o thread runs.
