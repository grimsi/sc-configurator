Tessellation:

Sets the tessellation quality. Higher value means better quality.

Pre-Render Tessellation:

Toggles pre-rendering for tessellation. Use only on high-end GPUs.

Hardware Tessellation Support:

Toggles hardware rendering for tessellation. Use on newer PCs.

Level of Detail:

Sets the general level of detail in the game. Low Medium High Very High

Multi-Core CPU Support:

Toggles multi-core support. Use on newer PCs.

Physics GPU:

Sets the GPU on which the physics are calculated (default: 1). Only change when using Multi-GPU systems. 1 2 2 4
